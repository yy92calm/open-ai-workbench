# packages/shared

Shared types and utilities used across the desktop app, SDK, and runtime tooling.

Includes the stable, versioned schemas that hold the system together:

- Artifact / manifest schema.
- Provenance event schema (`provenance.jsonl` records).
- Plan, tool-call, and approval-request types.
- Project workspace layout constants.

No UI and no I/O side effects — pure types and helpers only.
